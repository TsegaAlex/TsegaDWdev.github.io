Dimensional Modeling for Amazon Web Server

I designed and implemented a data warehouse for Amazon's sales platform using the full SQL Server BI stack (SSIS, SSAS, SSRS, and Power BI). OLTP data—covering products, categories, customers, and sales— was extracted, transformed, and loaded into a staging area, then into a star-schema-based OLAP warehouse. Dimension tables were incrementally loaded, and the FactSales table was populated with proper surrogate keys using lookups, including date and location references. I built a multidimensional cube using SSAS, wrote MDX queries for analysis, and developed reports with SSRS to deliver actionable business insights.
